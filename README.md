#  Motor Insurance Management System (MySQL)

##  Project Overview

The **Motor Insurance Management System** is a MySQL database project developed to manage motor insurance operations efficiently. It maintains records related to customers, brokers, vehicles, insurance products, quotations, policies, premium rates, payments, and accounting information. The project demonstrates core and advanced SQL concepts through a real-world insurance management scenario.

---

##  Objectives

- Design a normalized relational database for a motor insurance system.
- Manage customer, broker, vehicle, and policy information.
- Generate quotations and issue insurance policies.
- Maintain premium rates and accounting records.
- Demonstrate advanced SQL concepts using a real-world project.

---
